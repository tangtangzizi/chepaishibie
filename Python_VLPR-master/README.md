# Python_VLPR
## 首先需要知道的
毕业设计基于Opencv的车牌识别系统
- 稍微详细的解释见[博客链接](https://www.cnblogs.com/yinghualuowu/p/9182382.html)
- 版本：python3.7.3，opencv4.0.0.21，numpy1.16.2，tkinter和PIL5.4.1.
